# Security Policy

## Supported Versions

We provide security updates for the latest published version of `@letsprogram/ng-pdf-viewer`.

| Version        | Supported |
| -------------- | --------- |
| Latest         | Yes       |
| Older versions | No        |

## Reporting a Vulnerability

Please do not open a public GitHub issue for security vulnerabilities.

Instead, report vulnerabilities privately by emailing:

- letsprogram30@gmail.com

Please include the following details when possible:

- A clear description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested mitigation (if known)
- A proof-of-concept or screenshots/logs (if available)

## Response Expectations

- Initial acknowledgement: within 72 hours
- Triage and severity assessment: within 7 days
- Fix timeline: based on severity and complexity

We will keep you informed throughout the process and credit you in release notes (if you want) after the fix is published.

Thank you for helping keep the project and community safe.
